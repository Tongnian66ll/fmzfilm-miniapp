# 分秒帧影视 - 微信小程序

## 快速开始
1. 解压本项目
2. 打开「微信开发者工具」→ 导入项目 → 选择 miniapp 文件夹
3. AppID: `wx482755173e3708de`

## 开通云开发
1. 开发者工具 → 云开发 → 创建环境 `fmzfilm-prod`
2. 数据库建3个集合：`cases`、`bookings`、`quote_config`
3. 右键 cloud-functions 下每个文件夹 → 上传并部署
